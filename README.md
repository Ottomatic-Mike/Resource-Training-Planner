# Training Plan Manager

**AI-Assisted Training Planning for Engineering Teams**

A powerful web service that enables engineering managers to create comprehensive, AI-assisted training plans for their technical team members. Built with Node.js + Express and Docker for easy deployment.

## üöÄ Quick Start

### With Docker (Recommended)

```bash
# Clone and start
git clone https://github.com/Ottomatic-Mike/Resource-Training-Planning.git
cd Resource-Training-Planning
docker-compose up -d

# Access at http://localhost:3000
```

### With Node.js

```bash
# Clone and start
git clone https://github.com/Ottomatic-Mike/Resource-Training-Planning.git
cd Resource-Training-Planning/app
npm install
npm start

# Access at http://localhost:3000
```

## üåü Features

- **AI-Assisted Training Plans** - Competency assessment, goal suggestions, course discovery
- **Resource Management** - Team profiles, competencies, budgets, calendars  
- **Course Catalog** - Multi-provider course database
- **Reports & Analytics** - Budget tracking, progress monitoring, gap analysis
- **Embedded CORS Proxy** - Secure AI API calls without browser restrictions
- **Docker Support** - Easy deployment with Docker Compose

**New in v2.0.0:** Web service architecture with embedded CORS proxy eliminates all browser security restrictions.
